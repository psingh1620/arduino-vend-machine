# arduino-vend-machine

Simple vending machine code using an Arduino Mega 2560. 

Components:
- Ultrasonic Sensor (motion detection for coin)
- 4 360 Degree Rotation Servo Motors 
- Press Keypad for user input
- 16 x 2 LCD Screen
